# deltascf_f
The oscillator strength for DeltaSCF in the velocity gauge. 

## Dependency
This script is an addon to PySCF. You may find PySCF from GitHub. 
The PySCF release used to generate the data is 2.8.0. 

## Test
Currently this script has a built-in example. Just run
```bash
python3 oscillator_strength.py
```
